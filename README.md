# Simon Game

A browser-based implementation of the classic Simon memory game. The objective of the game is to remember and repeat an increasingly long sequence of colors. Each level adds a new color to the pattern, making the challenge progressively harder.

This project was built using HTML, CSS, and JavaScript to practice DOM manipulation, event handling, and game logic.

---

## How to Play

1. Press any key to start the game.
2. The game will highlight one color.
3. Click the same color button.
4. Each level adds a new color to the sequence.
5. Repeat the full sequence in the correct order.
6. If you click the wrong color, the game ends and you must restart.

---

## Features

- Interactive color buttons
- Random sequence generation
- Level progression system
- Sound effects for each button
- Game over detection and restart
